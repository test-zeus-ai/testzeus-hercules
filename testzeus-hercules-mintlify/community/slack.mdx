## Getting Help

### Before Asking for Help
1. **Search the Documentation**: Check the comprehensive documentation first
2. **Search Slack History**: Use Slack's search to find previous discussions
3. **Check GitHub Issues**: Look for existing issues and solutions
4. **Try the Examples**: Review the provided examples and use cases

### When Asking for Help
Include the following information:
- **Environment Details**: OS, Python version, Hercules version
- **Configuration**: Relevant environment variables and settings
- **Error Messages**: Complete error messages and stack traces
- **Test Code**: The Gherkin feature file or test code causing issues
- **Expected vs Actual**: What you expected vs what actually happened

### Example Help Request
```
**Environment:**
- OS: Ubuntu 22.04
- Python: 3.10.12
- Hercules: v1.2.0
- Browser: Chromium

**Issue:**
Getting timeout errors when running UI tests

**Error Message:**
```
TimeoutError: Waiting for element '#login-button' timed out after 30000ms
```

**Test Code:**
```gherkin
Given I am on the login page
When I click the login button
Then I should see the dashboard
```

**Configuration:**
```bash
BROWSER_TIMEOUT=30000
HEADLESS=true
```

**What I've Tried:**
- Increased timeout to 60000ms
- Verified element selector exists
- Tested with HEADLESS=false
```
