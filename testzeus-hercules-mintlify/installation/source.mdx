## Prerequisites

Before building from source, ensure you have:

- **Python 3.8+** (3.10+ recommended)
- **Git** for cloning the repository
- **Node.js** (for some development tools)
- **Build tools** for your platform

### Platform-Specific Build Tools

<Tabs>
  <Tab title="Windows">
    ```bash
    # Install Visual Studio Build Tools
    # Download from: https://visualstudio.microsoft.com/visual-cpp-build-tools/
    
    # Or install via chocolatey
    choco install visualstudio2019buildtools
    ```
  </Tab>
  <Tab title="macOS">
    ```bash
    # Install Xcode Command Line Tools
    xcode-select --install
    
    # Install Homebrew (if not already installed)
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    ```
  </Tab>
  <Tab title="Linux">
    ```bash
    # Ubuntu/Debian
    sudo apt-get update
    sudo apt-get install -y build-essential python3-dev git
    
    # CentOS/RHEL
    sudo yum groupinstall "Development Tools"
    sudo yum install python3-devel git
    ```
  </Tab>
</Tabs>

## Clone the Repository

Clone the Hercules repository from GitHub:

```bash
git clone https://github.com/test-zeus-ai/testzeus-hercules.git
cd testzeus-hercules
```

## Development Setup

### 1. Create Virtual Environment

Create an isolated development environment:

```bash
# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

### 2. Install Dependencies

Install development dependencies:

```bash
# Install in development mode
pip install -e .

# Install development dependencies
pip install -r requirements-dev.txt

# Install pre-commit hooks
pre-commit install
```

### 3. Install Browser Dependencies

Install Playwright browsers for testing:

```bash
playwright install
```

## Build Configuration

### Poetry Setup (Alternative)

If you prefer using Poetry for dependency management:

```bash
# Install Poetry
curl -sSL https://install.python-poetry.org | python3 -

# Install dependencies
poetry install

# Activate virtual environment
poetry shell
```

### Environment Configuration

Create a development configuration file:

```bash .env.dev
# Development settings
DEBUG=true
LOG_LEVEL=DEBUG

# LLM Configuration
OPENAI_API_KEY=your-dev-api-key
OPENAI_MODEL=gpt-4o

# Browser Configuration
BROWSER_TYPE=chromium
HEADLESS=false
BROWSER_TIMEOUT=60000

# Development paths
RESULTS_DIR=./dev-results
LOGS_DIR=./dev-logs
```

## Running Tests

### Unit Tests

Run the test suite to verify your installation:

```bash
# Run all tests
pytest

# Run with coverage
pytest --cov=testzeus_hercules

# Run specific test categories
pytest tests/unit/
pytest tests/integration/
```

### Integration Tests

Run integration tests with real browsers:

```bash
# Run browser tests
pytest tests/integration/browser/

# Run API tests
pytest tests/integration/api/

# Run with specific browser
BROWSER_TYPE=firefox pytest tests/integration/browser/
```

Build the documentation locally:

## Custom Builds

### Building Distribution Packages

Create distribution packages:

```bash
# Build source and wheel distributions
python -m build

# Check distribution
twine check dist/*

# Install from local build
pip install dist/testzeus_hercules-*.whl
```

### Custom Agent Development

Create custom agents by extending base classes:

```python custom_agent.py
from testzeus_hercules.core.agents.base_nav_agent import BaseNavAgent

class CustomAgent(BaseNavAgent):
    agent_name: str = "custom_agent"
    prompt = """# Custom Agent
    
    You are a custom agent that handles specific testing scenarios.
    """
    
    def __init__(self, *args, **kwargs):
        super().__init__(*args, **kwargs)
        # Custom initialization
```

### Custom Tool Development

Add custom tools to extend functionality:

```python custom_tool.py
from testzeus_hercules.core.tools.tool_registry import register_tool

@register_tool
def custom_validation_tool(page, expected_value):
    """Custom validation tool for specific use cases."""
    # Implementation
    return {"success": True, "message": "Validation passed"}
```

## Debugging

### Debug Configuration

Set up debugging for development:

```python debug_config.py
import logging
import os

# Enable debug logging
logging.basicConfig(level=logging.DEBUG)

# Set debug environment variables
os.environ['DEBUG'] = 'true'
os.environ['HEADLESS'] = 'false'
os.environ['BROWSER_TIMEOUT'] = '120000'
```

### Browser Debugging

Debug browser interactions:

```bash
# Run with visible browser
HEADLESS=false hercules test.feature

# Enable browser developer tools
BROWSER_ARGS="--auto-open-devtools-for-tabs" hercules test.feature

# Connect to browser debugging port
BROWSER_ARGS="--remote-debugging-port=9222" hercules test.feature
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Build Failures">
    Common build issues and solutions:
    ```bash
    # Clear pip cache
    pip cache purge
    
    # Reinstall dependencies
    pip install --force-reinstall -r requirements.txt
    
    # Update build tools
    pip install --upgrade pip setuptools wheel
    ```
  </Accordion>

  <Accordion title="Import Errors">
    Fix Python path issues:
    ```bash
    # Install in development mode
    pip install -e .
    
    # Check Python path
    python -c "import sys; print(sys.path)"
    ```
  </Accordion>

  <Accordion title="Browser Issues">
    Resolve browser-related problems:
    ```bash
    # Reinstall browsers
    playwright uninstall --all
    playwright install
    
    # Check browser installation
    playwright install --dry-run
    ```
  </Accordion>

  <Accordion title="Permission Issues">
    Fix file permission problems:
    ```bash
    # Fix permissions on Unix systems
    chmod +x scripts/*.sh
    
    # Run as administrator on Windows
    # Right-click Command Prompt -> "Run as administrator"
    ```
  </Accordion>
</AccordionGroup>
