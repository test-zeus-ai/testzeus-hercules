## System Requirements

### Minimum Requirements
- **Operating System**: Windows 10+, macOS 10.14+, or Linux (Ubuntu 18.04+)
- **Python**: 3.8 or higher
- **Memory**: 4GB RAM minimum, 8GB recommended
- **Storage**: 2GB free space for installation and test artifacts

### Recommended Requirements
- **Python**: 3.10 or higher
- **Memory**: 16GB RAM for complex test scenarios
- **Storage**: 10GB+ for extensive test suites and results
- **Network**: Stable internet connection for LLM API calls

## Browser Dependencies

Hercules supports multiple browsers and will automatically download browser binaries:

  <Tab title="Chromium">

## LLM Provider Setup

Before using Hercules, you'll need access to at least one LLM provider:

### Supported Providers

| Provider | Models | Function Calling | Recommended For |
|----------|--------|------------------|-----------------|
| **OpenAI** | GPT-4o, GPT-4o-mini | ✅ | Production use |
| **Anthropic** | Claude 3.5 Haiku+ | ✅ | High-quality reasoning |
| **Groq** | Llama, Mixtral | ✅ | Fast inference |
| **Mistral** | Mistral Large/Medium | ✅ | European compliance |
| **Ollama** | Local models 70B+ | ✅ | Privacy-focused |
| **Gemini** | Gemini Pro | ⚠️ | Deprecated |

<Warning>
  Ensure your chosen model supports function calling and agentic activities. Larger models (70B+ parameters) generally perform better for complex test scenarios.
</Warning>

## Verification

After installation, verify Hercules is working correctly:

```bash
# Check installation
hercules --version

# Run a simple test
echo 'Feature: Test
  Scenario: Check Google
    Given I navigate to "https://google.com"
    Then I should see "Google"' > test.feature

hercules test.feature
```
