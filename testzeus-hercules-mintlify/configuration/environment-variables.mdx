## Overview

TestZeus Hercules uses environment variables for configuration, allowing you to customize behavior without modifying code. Variables can be set in your shell, `.env` files, or CI/CD systems.

## LLM Configuration

### Primary LLM Provider

Configure your main LLM provider for the planner agent:

```bash
# OpenAI
OPENAI_API_KEY=your-api-key-here
OPENAI_MODEL=gpt-4o
OPENAI_BASE_URL=https://api.openai.com/v1  # Optional

# Anthropic
ANTHROPIC_API_KEY=your-api-key-here
ANTHROPIC_MODEL=claude-3-5-haiku-20241022
ANTHROPIC_BASE_URL=https://api.anthropic.com  # Optional

# Groq
GROQ_API_KEY=your-api-key-here
GROQ_MODEL=llama-3.1-70b-versatile
GROQ_BASE_URL=https://api.groq.com/openai/v1  # Optional

# Mistral
MISTRAL_API_KEY=your-api-key-here
MISTRAL_MODEL=mistral-large-latest
MISTRAL_BASE_URL=https://api.mistral.ai/v1  # Optional
```

### Agent-Specific LLM Configuration

Configure different models for specific agents:

```bash
# Planner Agent (main orchestrator)
PLANNER_AGENT_MODEL=gpt-4o
PLANNER_AGENT_PROVIDER=openai
PLANNER_AGENT_TEMPERATURE=0.1

# Browser Navigation Agent
BROWSER_NAV_AGENT_MODEL=claude-3-5-haiku-20241022
BROWSER_NAV_AGENT_PROVIDER=anthropic
BROWSER_NAV_AGENT_TEMPERATURE=0.0

# API Navigation Agent
API_NAV_AGENT_MODEL=llama-3.1-70b-versatile
API_NAV_AGENT_PROVIDER=groq
API_NAV_AGENT_TEMPERATURE=0.1

# Security Agent
SEC_NAV_AGENT_MODEL=gpt-4o
SEC_NAV_AGENT_PROVIDER=openai
SEC_NAV_AGENT_TEMPERATURE=0.0

# SQL Agent
SQL_NAV_AGENT_MODEL=claude-3-5-haiku-20241022
SQL_NAV_AGENT_PROVIDER=anthropic
SQL_NAV_AGENT_TEMPERATURE=0.0
```

### LLM Parameters

Fine-tune LLM behavior:

```bash
# Temperature (0.0-1.0, lower = more deterministic)
LLM_TEMPERATURE=0.1

# Maximum tokens per request
LLM_MAX_TOKENS=4000

# Request timeout in seconds
LLM_TIMEOUT=120

# Number of retries for failed requests
LLM_MAX_RETRIES=3

# Backoff factor for retries
LLM_BACKOFF_FACTOR=2
```

## Browser Configuration

### Basic Browser Settings

```bash
# Browser type: chromium, firefox, webkit
BROWSER_TYPE=chromium

# Run browser in headless mode
HEADLESS=true

# Browser timeout in milliseconds
BROWSER_TIMEOUT=30000

# Page load timeout in milliseconds
PAGE_LOAD_TIMEOUT=30000

# Element wait timeout in milliseconds
ELEMENT_TIMEOUT=10000
```

### Browser Arguments

Customize browser launch arguments:

```bash
# Additional browser arguments (space-separated)
BROWSER_ARGS="--no-sandbox --disable-dev-shm-usage --disable-gpu"

# User agent string
USER_AGENT="Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36"

# Window size for browser
WINDOW_WIDTH=1920
WINDOW_HEIGHT=1080

# Viewport size
VIEWPORT_WIDTH=1920
VIEWPORT_HEIGHT=1080
```

### Remote Browser Testing

Configure remote browser platforms:

```bash
# BrowserStack
BROWSERSTACK_USERNAME=your-username
BROWSERSTACK_ACCESS_KEY=your-access-key
BROWSERSTACK_BUILD_NAME="Hercules Test Run"
BROWSERSTACK_PROJECT_NAME="Test Automation"

# LambdaTest
LAMBDATEST_USERNAME=your-username
LAMBDATEST_ACCESS_KEY=your-access-key
LAMBDATEST_BUILD_NAME="Hercules Test Run"

# AnchorBrowser
ANCHOR_BROWSER_API_KEY=your-api-key
ANCHOR_BROWSER_PROJECT_ID=your-project-id

# Custom CDP URL for remote browsers
CDP_URL=wss://your-browser-provider.com/session
```

### Browser Capabilities

Configure specific browser capabilities:

```bash
# Chrome/Chromium specific
CHROME_BINARY_PATH=/usr/bin/google-chrome
CHROME_DRIVER_PATH=/usr/local/bin/chromedriver

# Firefox specific
FIREFOX_BINARY_PATH=/usr/bin/firefox
GECKO_DRIVER_PATH=/usr/local/bin/geckodriver

# WebKit specific
WEBKIT_BINARY_PATH=/usr/bin/webkit
```

## Loading Environment Variables

### From .env File

```bash
# Load from default .env file
hercules test.feature

# Load from specific .env file
hercules --env-file .env.prod test.feature
```

### From Shell

```bash
# Export variables in shell
export OPENAI_API_KEY="your-api-key"
export BROWSER_TYPE="chromium"
hercules test.feature

# Set variables for single command
HEADLESS=false hercules test.feature
```

### From Configuration File

```json config.json
{
  "environment": {
    "OPENAI_API_KEY": "your-api-key",
    "BROWSER_TYPE": "chromium",
    "HEADLESS": true
  }
}
```

```bash
hercules --config config.json test.feature
```

## Security Considerations

### Sensitive Data Protection

```bash
# Use environment variables for secrets
export OPENAI_API_KEY="$(cat ~/.secrets/openai_key)"

# Avoid logging sensitive data
export LOG_SENSITIVE_DATA=false

# Use secure storage for API keys
export USE_KEYRING=true
```

### Network Security

```bash
# Enable SSL verification
export VERIFY_SSL=true

# Use secure protocols only
export FORCE_HTTPS=true

# Configure proxy for secure networks
export HTTPS_PROXY="https://proxy.company.com:8080"
```
